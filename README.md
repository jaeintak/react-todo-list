# react-todo-list
A simple todo list web with input, show list, cross off, show only completed todos, show only active todos, delete all completed todos
functions deployed with Surge. 

## Getting Started
Install React JS
```
npx create-react-app todo-list
cd todo-list
npm start
```
Install Surge
```
npm install -g surge
cd todo-list
npm run build
```
